NAS 배포 단계
1

frequency 폴더 전체를 NAS에 복사

NAS의 웹 공유폴더 (또는 아무 공유폴더)에 복사합니다.

NAS공유폴더/frequency/ index.html server.py refresh.py start.bat data_wr.json ← refresh.py가 여기에 저장하게 됨 data500.json graveyard.json ← 자동생성됨
2

refresh.py의 OUT 경로 수정

JSON을 NAS 공유폴더에 저장하도록 경로를 바꿉니다.

# 기존 OUT = "G:/내 드라이브/work/frequency" # 변경 (NAS 경로 예시) OUT = "\\\\NAS이름\\frequency" # Windows UNC 경로 OUT = "Z:/frequency" # 또는 NAS를 드라이브로 매핑한 경우
3

NAS에서 server.py 실행

NAS에서 Python이 설치되어 있어야 합니다. Synology·QNAP은 패키지 센터에서 Python 3 설치 가능합니다.

# NAS SSH 접속 후 cd /volume1/frequency python3 server.py === Soltri 선생산 서버 === 로컬: http://localhost:8585 네트워크: http://192.168.x.x:8585
4

PC에서는 refresh.py만 실행

ERP 엑셀 다운로드 후 refresh.py를 실행하면 JSON이 NAS에 저장됩니다.
server.py는 NAS에서 계속 켜져 있으므로 PC에서 띄울 필요 없습니다.

# PC에서 python refresh.py ← 이것만 실행 # 이후 어느 기기에서든 http://192.168.x.x:8585 ← 접속
5

NAS IP 확인 방법

NAS 관리 페이지 또는 라우터에서 NAS에 할당된 IP를 확인합니다.
가능하면 NAS IP를 고정(예약)해두면 매번 달라지지 않습니다.

# 예시 NAS IP: 192.168.1.100 접속 주소: http://192.168.1.100:8585
