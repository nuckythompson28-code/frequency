📌 실제 사용 흐름

① ERP엑셀 다운로드
→
② start.bat더블클릭
→
③ refresh.py분석 + 시트 업로드
→
④ index.html자동으로 열림
→
⑤ 카드 클릭생산지시 중
→
⑥ Apps Script무덤 탭에 저장
⑦ ERP 발주안전재고 등록
→
⑧ 발주조회.xlsx다운로드
→
⑨ start.bat재실행
→
⑩ refresh.py날짜 매칭
→
⑪ 자동 이동생산지시 완료
→
⑫ 재고 상승완료 목록 제거

🖥️ 내 PC에서 하는 것

▸ start.bat 더블클릭 → ERP 갱신 + 대시보드 열기
▸ index.html 더블클릭 → 그냥 보기만 할 때
⚠ 서버 실행 불필요

📋 구글시트에 저장되는 것

▸ 선생산리스트 탭 — 분석 결과 (refresh.py가 씀)
▸ 주문이력 탭 — 주문/발주 이력 (refresh.py가 씀)
▸ 무덤 탭 — 생산지시 중/완료 (Apps Script가 씀)

⚡ Apps Script 역할

▸ 무덤 데이터를 구글 서버에서 읽고 씀
▸ JSONP 방식 → 서버 없이 파일에서 직접 호출 가능
▸ 다른 기기에서도 같은 무덤 상태 공유

🌐 index.html 동작

▸ 열리면 Apps Script에서 무덤 로드
▸ 구글시트에서 선생산 데이터 로드 (JSONP)
▸ 카드 클릭 → 즉시 Apps Script에 저장
▸ 로컬 캐시(localStorage)도 병행 저장
