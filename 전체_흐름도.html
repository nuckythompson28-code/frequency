2. 두 페이지 역할 비교

📊 안전재고발주.html

  • 선생산 필요 품목 카드 대시보드
  • G1/G2 그룹 필터 버튼
  • 생산지시 중 → 완료 → 재고충분 상태관리
  • 카드 클릭으로 생산지시, 되돌리기
  • PROFORMA INVOICE 엑셀 다운로드
  • Google Sheets 변경로그 (시간+PC)
  • 추적불가 품목 자동 감지
  • Apps Script 연동 (load/save/log)

🔍 라벨이력조회.html

  • 내경/외경/높이/소재 입력 검색
  • 현재고 크게 표시 (색상 코딩)
  • 최대 생산 추천량 (빨간색, 최대 500개)
  • 발송 이력 4열 카드
  • 왼쪽 선생산필요 품목 카드 리스트
  • 거래처별 통계
  • 입력칸 클릭 시 자동 선택
  • Google Sheets에서 라벨+재고 로드
